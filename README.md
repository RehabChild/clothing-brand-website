# RehabChild — The Healing Stage

The official website for **RehabChild**, a clothing brand built around the theme of healing and personal transformation.

## Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Landing page with hero, brand philosophy, product highlights, Instagram feed, and newsletter signup |
| Shop | `shop.html` | Full product catalog with cart functionality |
| Lookbook | `lookbook.html` | Visual showcase of the SS26 collection |
| About | `about.html` | Brand story and philosophy |
| Contact | `contact.html` | Contact form for inquiries |

## Tech

Built with plain HTML, CSS, and JavaScript — no frameworks required.

- `mobile.css` — responsive styling for smaller screens
- `cart.css` / `cart.js` — shopping cart functionality (shop page)
- Contact form and newsletter signup are powered by [Formspree](https://formspree.io)

## Running Locally

No build step needed. Clone the repo and open `index.html` in your browser:

```bash
git clone https://github.com/RehabChild/clothing-brand-website.git
cd clothing-brand-website
```

Then just open `index.html` directly, or serve it locally with:

```bash
python3 -m http.server 8000
```

and visit `http://localhost:8000`.
